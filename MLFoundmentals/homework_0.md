1.概率统计
1.3条件概率
P(A|B) = P(AB) / P(B)
所以:
在已知一个正面朝上的硬币条件下,三个硬币全都正面朝上的概率:
等于
这两个条件同时发生的概率1/8 除以 至少一个正面朝上的概率(1-1/8)
等于1/7

1.4 贝叶斯公式
P(A∩B) = P(A)*P(B|A)
        =P(B)*P(A|B)
==》P(A|B) = P(A)*P(B|A) / P(B)
答案P(x=-1 | |x|=1) = P(|x|=1 | x=-1) * P(-1) / P(|x|=1)
=1*1/2*1/4  /  (1/2*1/4 + 1/2*1/8) =2/3
